# Code Editor (Open Source)

## install

Step1. download source code

```git
git clone https://github.com/AkramStation/Code-Editor.git
```

Step2. install dependency

(Recommand)

```npm
npm install
```

or

```npm
npm install electron --save-dev
```

Step3. run app

```npm
npm starts
```
